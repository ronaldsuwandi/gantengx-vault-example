Initially it follows the system strictly but overtime it evolves and no longer uses decimals as described in [[🤖 System]]. Main purpose of this index is **not** to categorise my notes as that is handled by the note taking system. **This is purely for any file references**. If the note somehow have reference to Google docs or file, then the Google docs and file should be stored according to the index.

I still use index number but only up to the category/area level and not the id part

Any note references should be somehow linked through the [[🤖 System#MOC (Maps of Content)]]

1. iCloud Documents
2. Google Drive (link)
3. NAS
